myFlix App Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object containing an array with all movies
Get data about a single movie by title /movies/:Title GET none A JSON object containing data about a single movie title, including description, genre, director, image URL, whether it's featured or not. Example: { "Title": "Goldeneye", "Genre": { "Name": "action", "Description": "Action movies will contain a lot of violence and stunts." }, "Director": { "Name": "Martin Campbell", "Bio": "Born in a remote village in the Swiss Alps", "Birth": 1954, "Death": "N/A" }, "ImagePath": "https://www.imdb.com/title/tt0113189/mediaviewer/rm486027265/?ref_=tt_ov_i", "Featured": false }
Get data about a movie's genre /movies/genre/:genreName GET none A JSON object containing the genre name and description of genre. Example: { "Name": "Action", "Description": "Action movies will contain a lot of violence and stunts." }
Get data about a movie's director /movies/director/:directorName GET none A JSON object containing the directors bio and birth year. Example: { "Name": "Martin Campbell", "Bio": "Born in a remote village in the Swiss Alps", "Birth": 1954, "Death": "N/A" }
Add a new user /users POST A JSON object holding user data as such: { "Name": string, "Password": string, "Birthday": date, "FavoriteMovies": array } A JSON object holding the user name along with unique ID. Example: { "Name": "Mary", "Password": "maryG@gmail.com", "Birthday": "10-31-1990" "FavoriteMovies": [] }
Update user info /users:userName PUT A JSON object containing the chosen key/value pair to update. Example: { "UserName": string, "Password": string, "Email": string, "favoriteMovies": array } A JSON object containing all user information after being updated. Example: { "UserName": "kingkenny", "Password": "scottishroyalty", "Email": "kingkenny@gmail.com" "FavoriteMovies": [ ObjectId("6503933cd5ad7180cd9d51cf")] }
Add a movie to user's favorites list users/:UserName/movies/:movieID POST none A JSON object with the updated array of favorite movies: { "UserName": "Kim", "Password":"12345", "Email": "kim@gmail.com" "FavoriteMovies": [ ObjectId("6503933cd5ad7180cd9d51cf")] }
Remove a movie from user's favorites list /users/:UserName/movies/:movieID DELETE none A JSON object with favorite movie array updated. Example: { "UserName": "Joe", "Password": "abcdef", "Email": "joe@gmail.com" "FavoriteMovies": [] }
Delete user's account /users/:UserName DELETE none An updated users JSON object containing the remaining users. Example: { "UserName": "Kim", "Password": "12345", "Email": "kim@gmail.com" "FavoriteMovies": [] }
