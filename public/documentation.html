myFlix App Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object containing an array with all movies
Get data about a single movie by title /movies/:title GET none A JSON object containing data about a single movie title, including description, genre, director, image URL, whether it's featured or not. Example: { "title": "Goldeneye", "genre": { "name": "action", "description": "Action movies will contain a lot of violence and stunts." }, "director": { "name": "Martin Campbell", "bio": "Born in a remote village in the Swiss Alps", "birth": 1954 }, "imageURL": "https://www.imdb.com/title/tt0113189/mediaviewer/rm486027265/?ref_=tt_ov_i", "featured": false }
Get data about a movie's genre /movies/genre/:genreName GET none A JSON object containing the genre name and description of genre. Example: { "name": "action", "description": "Action movies will contain a lot of violence and stunts." }
Get data about a movie's director /movies/director/:directorName GET none A JSON object containing the directors bio and birth year. Example: { "name": "Martin Campbell", "bio": "Born in a remote village in the Swiss Alps", "birth": 1954 }
Add a new user /users POST A JSON object holding user data as such: { "name": "Mary", "favoriteMovies": [] } A JSON object holding the user name along with unique ID. Example: { "name": "Mary", "favoriteMovies": [], "id": "5aa59ee6-e5e3-4515-bb96-3ee3dc874e25" }
Update user info /users:id PUT A JSON object containing the chosen key/value pair to update. Example: { "name": "Kim Updated", "favoriteMovies": [] } A JSON object containing all user information after being updated. Example: { "id": 1, "name": "Kim Updated", "favoriteMovies": [] }
Add a movie to user's favorites list users/:id/:movieTitle POST none A JSON object with the updated array of favorite movies: { "id": 1, "name": "Kim", "favoriteMovies": "Goldeneye" }
Remove a movie from user's favorites list /users/:id/:movieTitle DELETE none A JSON object with favorite movie array updated. Example: { id: 2, name: "Joe", favoriteMovies: [] }
Delete user's account /users/:id DELETE none An updated users JSON object containing the remaining users. Example: { id: 1, name: "Kim", favoriteMovies: [] }
